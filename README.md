## О проекте

Открытый сборник задач по физике для школьников 10–11 классов и абитуриентов.  
Структура построена по принципу постепенного усложнения: каждая задача имеет **рейтинг сложности** (от 200 до 3000), что позволяет индивидуально подбирать материал под уровень ученика.

**Вдохновлено:** классическим сборником «3800 задач по физике, рейтинговой системой codeforces».

## 🚀 Быстрый старт

### Требования

- **MiKTeX** (или TeX Live) — [скачать](https://miktex.org/download)
- **VS Code** — [скачать](https://code.visualstudio.com/)
- **Расширение LaTeX Workshop** для VS Code

### Установка

1.  Клонируй репозиторий:
    ```bash
    git clone https://github.com/IronSamuray/physics-book.git
    cd physics-book
    ```

2.  Открой папку в VS Code.

3.  Открой файл `main.tex` и нажми `Ctrl + Alt + B` для компиляции.

4.  Готовый PDF появится в правой панели.
