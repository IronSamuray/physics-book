## 🎯 О проекте

Открытый сборник задач по физике для школьников 10–11 классов и абитуриентов.  
Структура построена по принципу постепенного усложнения: каждая задача имеет **рейтинг сложности** (от 200 до 3000), что позволяет индивидуально подбирать материал под уровень ученика.

**Вдохновлено:** классическим сборником «3800 задач по физике» и рейтинговой системой Codeforces.

## 🚀 Быстрый старт

### Требования

| Компонент | Ссылка |
|-----------|--------|
| **MiKTeX** (или TeX Live) | [Скачать](https://miktex.org/download) |
| **VS Code** | [Скачать](https://code.visualstudio.com/) |
| **Расширение LaTeX Workshop** | Установить из Marketplace VS Code |

### Установка

1. Клонируй репозиторий:
   ```bash
   git clone https://github.com/IronSamuray/physics-book.git
   cd physics-book
   ```

2. Открой папку в VS Code.

3. Открой файл `main.tex` и нажми `Ctrl + Alt + B` для компиляции.

4. Готовый PDF появится в правой панели.

---

## ⚙️ Настройка сборки

### Версия для ученика (без решений)

Открой `config/settings.tex` и измени настройки:

```latex
\solutionsfalse   % Решения скрыты
% \solutionstrue  % Закомментировано
```

### Версия для репетитора (с решениями)

```latex
\solutionstrue    % Решения показаны
% \solutionsfalse % Закомментировано
```

После изменения перекомпилируй `main.tex`.

---

## ✍️ Как добавить задачу

1. Открой нужный файл в папке `chapters/...` (или создай новый).

2. Используй шаблон:

   ```latex
   \begin{problem}{номер}
       Условие задачи...
       
       \rating{800}
   \end{problem}

   \begin{solution}
       Подробное решение с формулами...
   \end{solution}
   ```

3. Сохрани файл и скомпилируй проект.
